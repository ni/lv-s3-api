LV Interface for Amazon S3.lvlib:Get Bucket Website Configuration.vi

This operation returns the website configuration. This operation requires the S3:GetBucketWebsite permission. By default, only the bucket owner can read the bucket website configuration. Bucket owners can allow other users to read the website configuration by writing a bucket policy granting them the S3:GetBucketWebsite permission.

Bucket

Bucket specifies the name of the bucket whose Website Configuration you want to get.

error in (no error)

WebsiteConfiguration contains information about the Website Configuration of the bucket.

      Suffix

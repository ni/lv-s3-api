LV Interface for Amazon S3.lvlib:LL_GET Service.vi

Note: This VI is no longer supported. Use the Low-Level HTTP Request AWS v4 VI instead.

This implementation of the GET operation returns a list of all buckets owned by the authenticated sender of the request. Use this VI instead of the List Buckets.vi if you need more iformation than that returned by that VI.

Call Result

Call Result returns the result of the call. If the call completed successfully, this output returns "Success", otherwise it will return an error.

ListAllMyBucketsResult

ListAllMyBucketsResult contains information about the owner and the buckets he or she owns.

    Owner

    Owner contains elements that identifies the buckets' owner.

      ID

      ID is the Canonical User ID of the owner.

      DisplayName

      DisplayName is the Principal's screen name (Email address).

    Buckets

    Buckets array is a list of all the buckets that the authenticated user of the request owns.

      Bucket

      Bucket contains information about a specific bucket.

        Name

        Name contains the name of the bucket.

        CreationDate

        CreationDate shows the date and time (in UTC) the bucket was created.

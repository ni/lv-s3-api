LV Interface for Amazon S3.lvlib:Get Bucket Notification Configuration.vi

This operation returns the notification configuration of a bucket. If notifications are not enabled on the bucket, the operation returns an empty NotificatonConfiguration element. You must be the bucket owner to read the notification configuration of a bucket. The bucket owner can use a bucket policy to grant permission to other users to read this configuration with the s3:GetBucketNotification permission.

Bucket

Bucket specifies the name of the bucket whose Notification Configuration you want to get.

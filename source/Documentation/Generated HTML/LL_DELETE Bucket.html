LV Interface for Amazon S3.lvlib:LL_DELETE Bucket.vi

Note: This VI is no longer supported. Use the Low-Level HTTP Request AWS v4 VI instead.

This core implementation of the DELETE operation may be used to implement special requests or API calls not present in this toolkit.

Bucket

Bucket is the name of the bucket where you want to send the DELETE request.

error in (no error)

error in can accept error information wired from VIs previously called. Use this information to decide if any functionality should be bypassed in the event of errors from other VIs.

Right-click the error in control on the front panel and select Explain Error or Explain Warning from the shortcut menu for more information about the error.

    status

    status is TRUE (X) if an error occurred or FALSE (checkmark) to indicate a warning or that no error occurred.

    Right-click the error in control on the front panel and select Explain Error or Explain Warning from the shortcut menu for more information about the error.

    code

    code is the error or warning code.

    Right-click the error in control on the front panel and select Explain Error or Explain Warning from the shortcut menu for more information about the error.

    source

    source describes the origin of the error or warning.

    Right-click the error in control on the front panel and select Explain Error or Explain Warning from the shortcut menu for more information about the error.

Path and query string

Path and query string is the path and query part of the URI where the request is going to be sent. This string has to start with a "/" which represents the root of the bucket's URI.

Default value: /

Headers

Headers array is the list of HTTP Request headers to send with the HTTP Request.

    HTTP Header-Value pair

    HTTP Header-Value pair represents a HTTP header by its name and value.

      Header

      Header specifies the header field to add to the HTTP request. The Header control provides a pull-down menu with some predefined, common header fields.

      Value

      Value specifies the value of the HTTP header.

Call Result

Call Result returns the result of the call. If the call completed successfully, this output returns "Success", otherwise it will return an error.

LV Interface for Amazon S3.lvlib:Set AWS Credentials.vi

Stores the AWS Credentials (AWS Access Key & AWS Secret Key) in a FG. You need to call this VI only once, prior to any Amazon S3 requests. The credentials will be used during all the requests you make to Amazon S3. To obtain AWS Credentials you must first register with Amazon AWS. Because this operation initializes the FG that stores your credentials, you should place this VI in your Top-level VI, or at a place in your VI hierarchy so that you can ensure the FG will not go out of memory while you are using Amazon S3 toolkit VIs.

AWS Credentials in

AWS Credentials in specifies the AWS Credentials that will be used during subsequent S3 API Calls, until this VI is called again with other Credentials.

    Access Key

    Access Key specifies the Access Key ID that is registered with Amazon S3.

    Secret Key

    Secret Key ID is used to authenticate the user with the specified Access Key.

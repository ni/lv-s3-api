LV Interface for Amazon S3.lvlib:Set Bucket RequestPayment Configuration.vi

This operation sets the RequestPayment configuration of a bucket. By default, the bucket owner pays for downloads from the bucket. This configuration parameter enables the bucket owner (only) to specify that the person requesting the download will be charged for the download.

RequestPayment Configuration

RequestPaymentConfiguration specifies the RequestPayment Configuration to use for the bucket.

    Payer

    Payer specifies who pays for the download and request fees. Valid Values: Requester, BucketOwner

    Default value: BucketOwner

Bucket

Bucket specifies the name of the bucket whose RequestPayment Configuration you want to set.

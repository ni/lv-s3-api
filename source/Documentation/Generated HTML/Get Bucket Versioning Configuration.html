LV Interface for Amazon S3.lvlib:Get Bucket Versioning Configuration.vi

This operation returns the versioning state of a bucket. To retrieve the versioning state of a bucket, you must be the bucket owner. This implementation also returns the MFA Delete status of the versioning state.

VersioningConfiguration

VersioningConfiguration contains the current Versioning Configuration of the bucket.

    Status

    Status shows whether the Versioning capability of the bucket is Enabled.

    Valid values:

    - Enabled—Versioning is enabled for the objects in the bucket.

    All objects added to the bucket receive a unique version ID.

    - Suspended—Versioning is diabled for the objects in the bucket.

    All objects added to the bucket receive the version ID null.

    - Disabled—If the versioning state has never been set on a bucket, it has no versioning state; so Disabled is returned as the Versioning Status.

    MfaDelete

    MfaDelete shows whether MfaDelete option is enabled for the bucket.

    If the MFA Delete status is enabled, the bucket owner must use an authentication device to change the versioning state of the bucket.

LV Interface for Amazon S3.lvlib:Delete Object.vi

The operation removes the null version (if there is one) of an object and inserts a delete marker, which becomes the latest version of the object. If there isn't a null version, Amazon S3 does not remove any objects.

Auth Device Data

Auth device data has to be provided to authenticate the request, if Mfa Delete is enabled on the bucket.

    SerialNumber

    Serial number of your Authentication device (token)

    TokenCode

    The Value currently displayed on your Authentication device (token).

Bucket

Bucket specifies the name of the bucket where the object you want to delete resides.

Object Path

Object Path is the path to the object you want to delete, relative from the bucket's root.

Note: leading '/' is not necessary, but will still work.

Version ID

Version ID can be used to remove a specific version of the object, if Versioning is enabled on the bucket.

If no Version ID is specified, the latest version of the object will be removed.

Default value: empty string - Use the latest version

Region

Region specifies the geographical location of the bucket.

Versioning Data

Versioning Data contains information about the executed Delete operation, if Versioning is enabled on the bucket.

    Version ID

    Version ID is the Version ID of the delete marker created as a result of the DELETE operation. If you delete a specific object version, the value returned by this header is the version ID of the object version deleted.

    Delete marker

    Delete marker specifies whether the versioned object that was permanently deleted was (TRUE) or was not (FALSE) a delete marker.

    In a simple DELETE, this header indicates whether (TRUE) or not (FALSE) a delete marker was created.

LV Interface for Amazon S3.lvlib:Get Bucket ACL.vi

This operation returns the access control list (ACL) of a bucket. To use it, you must have READ_ACP access to the bucket. If READ_ACP permission is granted to the anonymous user, you can return the ACL of the bucket without using an authorization header.

AccessControlPolicy

AccessControlPolicy contains information about the bucket and the ACL associated with it.

    Owner

    Owner contains information about the bucket owner.

      ID

      ID is the Canonical User ID of the owner.

      DisplayName

      DisplayName is the screen name (Email address) of the bucket owner. This is returned only if the owner's email address (or the forum name, if configured) can be determined from the ID.

    AccessControlList

    AccessControlList array lists the ACL entries associated with the bucket.

      Grant

      Grant is a container for the grantee and his or her permissions.

        Grantee

        Grantee is the subject whose permissions are being set.

          ID

          ID is the Canonical User ID of the Grantee.

          DisplayName

          DisplayName is the screen name (Email address) of the Grantee.

        Permission

        Permission specifies the permission (FULL_CONTROL, WRITE, WRITE_ACP, READ, READ_ACP) given to the Grantee.
